# fretty.app

An interactive fretboard exploration tool for all stringed instruments.


## Project setup
```
npm install
```

### Run local development server
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files

```
npm run lint
```
